About Me

My Name is Justin Flanagan and I am a self-taught web developer. I first learned coding while minoring in computer science at college, working mostly with Java and Python. After graduating, I began teaching myself Javascript and found it much more enjoyable to work with a front-end language. Particularly, I love being able to create applications that I can view in a web browser and interact with. What I find most enjoyable about programming is how it enables me to expand or reinforce my knowledge when fixing errors, and the sense of accomplishment that comes from seeing an application perform as I envisioned.
